#NOStream#

Java library that avoid use of Stream interface.

* Current classes: <br />
   * Collection, List, ArrayList, LinkedList extends the counterpart Java structures . <br />
* Fuctions are: <br />
   * all
   * any
   * count
   * distinct
   * filter
   * filterIndexed
   * filterNotNull
   * first
   * firstOrNull
   * forEachIndexed
   * forEachReverse
   * groupBy
   * intersection
   * last
   * lastOrNull
   * map
   * mapIndexed
   * maxBy
   * minBy
   * orderBy
   * orderDecreasingBy
   * reduce
   * reduceReverse
   * reverse
   * take
   * takeLast
   * union
   * zipIndexed
   * zipWith
* How to use: <br/>
&emsp;    Import jar in your project then replace java.util.[Collection name] with structures.[Collection name]
